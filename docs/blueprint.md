# **App Name**: Xacto Hub

## Core Features:

- Project Showcase: Display the company's web development projects with descriptions and images.
- Technology Stack: List the technologies used in projects with brief explanations.
- Testimonials Section: Display client testimonials to build trust and credibility.
- Featured Companies: Showcase logos of companies Xacto has worked with.
- Contact Form: Allow potential clients to submit project inquiries through a contact form.
- New Project Submission: Authenticated users can submit project data and images to Firebase, and a tool will examine this input before incorporating it into the landing page.
- Firebase Integration: Connect the website to Firebase to store project data and images.

## Style Guidelines:

- Primary color: Dark teal (#008080) to convey professionalism and sophistication.
- Background color: Very light desaturated teal (#F0FFFF) to provide a clean, modern backdrop.
- Accent color: Deep cyan (#008B8B) for highlights and call-to-action buttons.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, for a contemporary feel.
- Use minimalist, custom icons to represent technologies and services.
- Modern, clean layout with a focus on whitespace and clear section divisions.
- Subtle animations for page transitions and loading states to enhance user experience.
